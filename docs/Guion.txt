**Habla cuando ponga la diapositiva de las tecnologías usadas**
Para nuestro sistema de información MediFlow empleamos python y el framework flask para el backend, para el frontend usamos html/css, bootstrap 5 y jinja2 como control
de plantillas y finalmente como motor de base de datos usamos mariadb. Tecnologías de punto en el mercado de software actual.
